Taller de arreglos unidimensional, bidimensional y ArrayList

Descripción del taller

Este taller tiene como objetivo aplicar los conceptos fundamentales de programación en Java, enfocados en el uso de:

Arreglos unidimensionales, Arreglos bidimensionales (matrices), ArrayList

A lo largo del desarrollo se implementan diferentes ejercicios que permiten reforzar habilidades como:

Recorridos de estructuras de datos

Búsqueda y ordenamiento

Manipulación de matrices

Uso de programación orientada a objetos (POO)

Aplicación de buenas prácticas como Clean Code y principios de programación (KISS, DRY)

Cada ejercicio fue desarrollado en un archivo independiente, facilitando su comprensión, ejecución y mantenimiento.

 Instrucciones para ejecutar el programa

Requisitos previos:

Tener instalado Java JDK (versión 8 o superior)

Tener un entorno de desarrollo

Clonar o descargar el proyecto

Abrir la carpeta del proyecto en el entorno de desarrollo

Ubicar el archivo que desea ejecutar, por ejemplo:

Ejercicio1Primos.java

Ejercicio2Pares.java

...

Ejercicio15IntercambiarFilas.java

Ejecutar el archivo desde el método main

Cada ejercicio es independiente y puede ejecutarse por separado.

Para el ejercicio de partidos: Asegurarse de que el archivo PartidoLiga.txt se encuentre en la misma ruta del proyecto
